### 使用GitHub

* 目的
  * 借助github 托管项目代码



### 基本概念

* 仓库（Repository）
  * 仓库用来存放项目代码，每个项目对应一个仓库， 多个开源项目则有多个仓库
* 收藏（star）
  * 收藏项目，方便下次查看
* 复制克隆项目（Fork）
  * 复制一个项目，同时显示来源，该项目独立存在

* 发起请求（full request）
  * 将复制的项目发送给 另一个项目，请求合并

* 关注（Watch）
  * 关注项目
* 事物卡片（lssue）
  * 发现代码Bug， 但时目前没有成型的代码，需要讨论时使用

* Github 主页
  * 登录创建账号
* 仓库主页
  * 显示项目信息，如项目代码，版本，收藏，关注，fork情况
* 个人主页
  * 个人信息，关注的人， 关注的库，自己的开源项目，贡献的开源项目



### 创建仓库

* 一个git库（仓库），对应一个开源项目
  * README.md 项目详细描述
  * LICENSE 协议
  * .gitignore 在使用Git操作时，忽略的文件
